## How to run this dapp

- clone the project

```bash
git clone https://github.com/bobeu/build-for-lisk
```

```bash
cd backend
yarn install
```

```bash
yarn compile
```

- deploy to lisk testnet
```bash
yarn testnet
```

- deploy to lisk mainnet
```bash
yarn mainnet
```

### Frontend

```bash
cd frontend
yarn install
```

- Development Mode
```bash
yarn run dev
```

- Production Mode
```bash
yarn run build
```

```bash
yarn start
```
